mTETRIS

This is the main Digital Coda website, a repository for web applications and web sites produced by its principal. The principal of this web site has over 10 years of experience using primarily Perl/Shell/SQL languages on various Linux/Unix platforms.

Most of the above experience has been supporting financial applications on multi-tiered web and n-tiered client/server environments.

The principal is also a Sun Certified Java Programmer.

  • DCQuery is a online demo stock query application, that allows mock lookups of a stock order based on price and quantity. The application uses Perl/MySQL on the backend.

  • WebMarks is a Perl/CGI bookmarking application sortable by alphabetic range, overall "DATE", and "Date Added" under alphabetic range selected.

  • The application has been rewritten in the Python/Bottle micro-web framework and again in the Perl/Mojolicious micro framework.

  • ChatBox is a demo Web 2.0 application that allows multiple parties to login and converse in chatrooms. The application is driven by a Perl/MySQL backend.

  • expressChat is a backend rewrite of ChatBox, that has now been rewritten in Javascript/expressJS.

  • PollCenter is a mock poll site that has been enhanced with third party plugins. If using IE, versions 7 and above are needed to properly view the site.

  • Tetris, this version, is written by Alex Fritze and is just slightly modded.
